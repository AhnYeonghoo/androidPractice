코틀린에 존재하는 특이한 자료형/키워드

1. Unit
- 자바로는 void
- 반환값이 없는 메서드에 사용


2. Any/Any?
- Any -> non-null 타입의 조상(Int, Short, ...)
- Any? -> nullable 타입의 조상 (Int?, Short?, ...)
- 모든 타입의 조상 클래스인듯?

3. Nothing
- fun 함수명(변수명: 타입, ...): Nothing {'
'   함수내용 -> "너 뭔가 잘못했다"
}
- 이 함수를 호출하면 뭔가 잘못되었다.
- 명시적으로 함수가 잘못됨을 알려줌(유지보수 용이인듯?)
