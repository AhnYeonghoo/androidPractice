예외처리
- 예외가 발생하는 부분 -> try { }
- 예외를 처리하는 부분 -> catch() { }
try {
    code -> 예외 발생 가능한 코드
} catch(변수명: ExceptionType) {
    예외처리 코드
} catch(변수명: ExceptionType2) {
    예외처리 코드
} finally {
    이러나 저러나 무조건 실행되어야 하는 코드
}
try문에서 A라는 예외가 발생하면 catch문에서 A라는 예외를 처리하겠다고 명시적으로 적어줌
그렇지 않으면 오류가 발생한다.
- 모든 종류의 예외를 포함하는 최고 조상 예외 타입 -> Exception
- try, catch는 각자 독립적으로 사용할 수 없다.
